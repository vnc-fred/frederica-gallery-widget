
# Widget Frederica

Este é um widget que mostra uma galeria tipo Instagram com base numa base de dados Notion.

## Como publicar no Vercel

1. Vai até https://vercel.com
2. Faz login com GitHub, Google ou email
3. Cria um novo projeto e faz upload destes ficheiros
4. Nomeia como `frederica-gallery`
5. Clica em "Deploy" — o link estará disponível em poucos segundos

## O que o widget faz

- Puxa as imagens do campo `Criativo` da tua base Notion
- Mostra-as em 3 colunas, proporção 4:5
- Ao clicar, abre a página no Notion

Qualquer dúvida, é só chamar!
